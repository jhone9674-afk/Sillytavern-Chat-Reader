# Sillytavern-Chat-Reader

A straightforward extension that allows you to read the Chat of your Sillytavern Characters without opening it.

<img width="104" height="31" alt="image" src="https://github.com/user-attachments/assets/554e232b-e665-4d69-9e1b-cddf90e1c196" />

All you have to do is simply click on this button, and you will be able to read your chats without loading them.

<img width="104" height="31" alt="image" src="https://github.com/user-attachments/assets/fbb8a0ab-77ac-498a-80ce-4138966db44a" />

Use ST's built-in extension installer with this URL:
https://github.com/jhone9674-afk/Sillytavern-Chat-Reader
